## 🎮 게임 소개
당신에게 남은 것은 단돈 **10만 원**과 **2주**의 시간!
매일 식판에 3가지 메뉴를 담아 식사를 해결해야 합니다. 하지만 조심하세요. 영양 불균형은 건강을 해치고, 맛없는 음식은 멘탈을 파괴합니다.

### 주요 특징
- **전략적 자원 관리:** 돈(Money), 멘탈(Mental), 건강(Health) 3가지 스탯 관리
- **다양한 메뉴:** 가성비 '유통기한 빵'부터 럭셔리 '스테이크'까지 12종의 음식
- **영양 밸런스 시스템:** 탄수화물, 단백질, 당류 등의 조합에 따른 버프/디버프 구현
- **랜덤 이벤트:** 2주 차(Day 8)에 찾아오는 부모님의 용돈 찬스 (확률 40%)
- **픽셀 아트 그래픽:** 레트로 감성의 귀여운 음식 아이콘 적용

## 🕹️ 플레이 방법
1. **메뉴 선택:** 원하는 음식 3가지를 골라 식판을 채웁니다.
2. **제한 시간:** **10초** 안에 메뉴를 고르지 못하면 패널티가 주어집니다.
3. **게임 오버 조건:** 잔고, 멘탈, 건강 중 하나라도 **0 이하**가 되면 게임 오버!
4. **승리 조건:** 14일 동안 생존에 성공하면 졸업(Clear)!

## 🛠️ 설치 및 실행 방법 (How to Run)

이 게임을 실행하려면 컴퓨터에 **Python(파이썬)** 이 설치되어 있어야 합니다.

1. 이 페이지 오른쪽 위의 초록색 <> Code 버튼을 누릅니다.  
2. **Download ZIP**을 클릭하여 다운로드 받은 뒤, 압축을 풉니다.  
3. VS코드에서 압축 푼 폴더를 열고, 터미널(CMD)을 엽니다(Ctrl+`).  
4. 필수 라이브러리를 설치합니다.  
```bash

# 터미널에 입력
pip install pygame

```
5. main.py 파일에 들어가 실행버튼을 누릅니다.
